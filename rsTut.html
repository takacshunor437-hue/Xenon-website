Random Sandbox Tutorials

First of all: What is this game??
So, basically, you gotta build. let's first talk about blocks.
There are 4 different basic blocks.
First: Stone
Second: wooden plank
Third: dirt (I think third is dirt.. right???)
And then fourth: Wooden trunk. Now, you can switch between these by pressing 1, 2, 3, or 4
1 is obviously stone, 2 is plank, 3 is dirt and 4 is wooden trunk.


Push mode:
Update apha 0.4 brought push mode into the game.
It's function is very simple: Objects don't push you away anymore (they don't collide).
Instead, the player pushes the object away.

How to activate push mode??
You kinda just press F. There's a black text in the games
bottom-left corner that basically tells you if push mode
is turned off or on.

Now, let's talk about that little cross on the screen
-So, basically it acts as a mouse cursor.
You control it with the arrow keys.
You place down blocks with Control (either left or right, both work)
and you delete blocks with Shift (again, either works).
!!IF the image of the cursor (the small cross) didn't load
it's a small black square instead.


OVERALL:
-You can clone yourself with C.
-You can move up with W.
-You can move down with S.
-You can move left with A.
-You can move right with D.


-You can open your inventory with I and navigate through it with the in-game cursor.
-You can delete spawned NPC's by interacting with them with the in-game cursor.
-your in-game cursor is a cross or a black square
if the image couldn't load for some reason.
-You can move this cursor with the Arrow keys.
-You can place blocks with either left or right Control.
-You can remove blocks with either left or right Shift.
-You can switch between basic blocks with 1,2,3,4
