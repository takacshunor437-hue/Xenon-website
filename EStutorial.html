Eternal Survival Tutorial text

First of all, there's countless things that might not be obvious
to the avarage PC player.
For example, controls:
Now, these buttons for running or aiming are not very common
in other games, but here, I tought using these keys are a lot more
comfortable for the player.
Moving:
WASD
Running (While pressing W or S): Q
Aiming with any weapon (It actually improves shooting accuracy):
E
To clear your save, there's no UI for it, because:
pressing three keys is a lot more comfortable than pressing ESC and then
selecting like five options.
And because it saved time
You can clear your save by pressing the key combination:
T-Z-U

What might not be clear for you:
How to solve and "type in" codes you have to solve in the game?
See, if you solved a code, you obviously have to find a device you can
type the code into.
After you found a solution, no matter if it's numbers or letters,
press the keys that might be one of the digits of the solved code, while
clicking on the device you type the code into.
For example, if there was a bomb with the code: A-D-B-C,
You would have to press A, D, B and C while clicking on the bomb.
This would activate the bomb and then detonate it.
You can also set mouse sensitivity with the arrow keys:
Arrow pointing upwards = higher mouse sensitivity
Arrow pointing downwards = lower mouse sensitivity
-PLUS!
If you ever need to stand on something that you can push,
you ALWAYS have to go near it and click on it.
This will immediately activate it's "Walkable" mode, and now
you'll be able to stand on it (If you are stuck on the first map with a
door covered with immovable boxes, this is incredibly useful! (you
push the crates to the position you want them to be, click while you're near them,
walk into them and BOOM you stepped on them. - you can obviously disable this by clicking again)

Now, you know everything about the game. (that might not be obvious)
Note that this is not a game that just "Tells you everything" - solve it.
That's what it was made for.
Enjoy the game!
